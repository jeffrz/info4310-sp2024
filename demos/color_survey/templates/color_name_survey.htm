Name a Color!

This is an informal survey for INFO4310 at Cornell University. In order to practice working with color information, we are collecting data about color names and favorite colors so that we can visualize them. This survey is inspired by the XKCD color survey.
The survey is anonymous, and no compensation is offered for completing it. Collected data will not be used for research, and will only be used for class activities.

You can take this survey as many times as you like. Each time you press submit you'll get a new color to name!
